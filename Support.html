Need Help?

This page answers some common questions about using Recipe Finder.

How do I log in?

Enter any username and password on the login screen, and then select the Login button.

How do I search for a recipe?

Use the search box on the home page. You can search by recipe name or by an ingredient.

How do I add a recipe?

Enter the recipe name, ingredients, and instructions. Then select the Add Recipe button.

Why did my added recipe disappear?

This basic version stores added recipes only while the page is open. Refreshing or closing the page resets the recipe list.
