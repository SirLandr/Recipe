Need Help?

This page answers common questions about using Recipe Finder.

How do I log in?

Enter any username and password on the login screen, and then select the Login button.

How does Recipe of the Day work?

The website automatically selects one recipe from the recipe list each day.

How do I search for a recipe?

Use the search box on the home page. You can search by recipe name or by ingredient.

How do I add a recipe?

Enter the recipe name, ingredients, and instructions. Then select Add Recipe.

Why did my added recipe disappear?

This basic version saves added recipes in the browser. Clearing browser data may remove them.
